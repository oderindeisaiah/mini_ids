# Mini Intrusion Detection System (IDS)

A lightweight Python-based IDS that analyzes authentication logs
to detect common attack patterns.

## Detection Rules
- Brute-force attacks (single IP, multiple failures)
- Credential stuffing (single user, multiple IPs)

## Security Concepts
- Intrusion detection
- Log correlation
- Threat modeling
- Blue team defense

## Technology
- Python
- Log analysis
- Security monitoring
